# my-stock-team 📊

> **한 줄 요약:** 종목 이름만 입력하면, 5명의 "AI 애널리스트"가 팀을 이뤄 한국 주식을 분석하고
> **발표용 파워포인트(PPT) 리포트**까지 자동으로 만들어 주는 도구입니다.
>
> ⚠️ **학습·연습용입니다. 실제 투자 자문이 아닙니다.** ("이 주식 사세요/파세요" 같은 말은 하지 않습니다.)

---

## 🤔 이게 뭔가요? (비개발자를 위한 설명)

여러분이 "삼성전자 분석해줘"라고 입력하면, 사람 애널리스트 팀이 일하듯이
**역할이 다른 AI 5명**이 각자 맡은 부분을 조사하고, 그 결과를 하나로 합쳐
**보기 좋은 발표 자료(PPT/PDF)**로 만들어 줍니다.

마치 증권사 리서치팀을 통째로 빌려 쓰는 느낌이라고 생각하시면 됩니다.

| AI 팀원 | 하는 일 | 비유하자면 |
|---|---|---|
| 🔴 **펀더멘털 분석가** | 회사 재무제표·실적·공시 분석 (매출, 영업이익 등) | 회사 가계부를 들여다보는 회계사 |
| 🟢 **시장·기술 분석가** | 주가 흐름, 이동평균선, 52주 최고/최저, 거래량 | 주가 차트를 읽는 트레이더 |
| 🔵 **뉴스·심리 분석가** | 관련 뉴스, 이슈, 시장 분위기 조사 | 신문·여론을 살피는 기자 |
| 🟣 **리스크 매니저** | 위 결과를 종합하고 위험 요소를 점검 | 팀을 총괄하는 팀장 |
| 🟡 **검수 담당** | 완성된 리포트에 오류·과장이 없는지 검사 | 출판 전 교정을 보는 편집자 |

이 5명이 협업해서 **표지 → 재무 → 차트 → 리스크 → 종합의견** 순서의
5부 구성 리포트를 만들어 냅니다.

## 📦 무엇이 필요한가요? (준비물)

이 도구는 **Claude Code** 라는 프로그램 안에서 작동하는 "플러그인(추가 기능)"입니다.
스마트폰 앱을 깔듯이, Claude Code에 이 기능을 추가해서 쓰는 방식입니다.

준비물은 다음과 같습니다.

1. **Claude Code** — 이 도구를 실행할 프로그램. (이미 이 글을 보고 계시다면 보통 설치되어 있습니다.)
2. **무료 데이터 키 1개 (`DART_KEY`)** — 회사 재무 정보를 가져올 때 필요합니다.
   - 발급처: https://opendart.fss.or.kr (금융감독원, **무료**)
   - 회원가입 → 인증키 신청하면 바로 받을 수 있습니다.
   - ⚠️ 이 키는 **비밀번호 같은 것**이라 절대 남에게 공유하거나 코드에 직접 적으면 안 됩니다. (아래 설정 방법 참고)
3. **약간의 프로그램 설치** — 파워포인트를 만들기 위한 도구들 (`python-pptx`, `matplotlib`).
   - 잘 모르시면 이 부분은 도와줄 사람에게 "사전 요건 설치"를 부탁하시면 됩니다.

## 🚀 설치 방법 (따라 하기)

Claude Code 안에서 아래 명령어를 **한 줄씩 순서대로** 입력하면 됩니다.
(명령어는 `/` 로 시작합니다. 입력창에 그대로 치면 돼요.)

### 방법 A — GitHub에서 바로 설치 (추천)

```
/plugin marketplace add seohee-lee-kb/my-stock-team
/plugin install my-stock-team
```

### 방법 B — 내 컴퓨터에 받아둔 폴더로 설치

```
/plugin marketplace add /폴더경로/my-stock-team
/plugin install my-stock-team
```

설치가 끝나면 준비 완료입니다.

## 🔑 비밀 키(DART_KEY) 설정하기

재무 분석을 하려면 위에서 발급받은 키를 등록해야 합니다.
플러그인 폴더 안에 **`.env`** 라는 이름의 파일을 만들고, 그 안에 아래 한 줄을 적으세요.

```
DART_KEY=여기에_발급받은_본인_키를_붙여넣기
```

> 💡 **걱정 마세요:** 이 `.env` 파일은 GitHub에 자동으로 **올라가지 않도록** 막혀 있습니다.
> 여러분의 비밀 키가 외부에 노출되지 않습니다.

## 💬 사용 방법 (실제로 써보기)

설치가 끝났으면, Claude Code 입력창에 아래처럼 입력해 보세요.

### 1. 전체 리포트 한 번에 만들기

```
/my-stock-team:stock-report 삼성전자
```
👉 분석 → 검수 → **파워포인트(PPT)와 PDF 리포트**까지 한 번에 만들어 줍니다.

### 2. 이미 만든 리포트만 검사하기

```
/my-stock-team:stock-verify 삼성전자
```
👉 이미 만들어 둔 리포트에 오류·과장이 없는지 검수만 합니다.

### 3. 그냥 말로 물어보기

명령어를 외울 필요 없이, **평소 말하듯** 질문해도 알아서 담당 AI에게 넘겨줍니다.

```
"삼성전자 주가 흐름 좀 봐줘"      →  🟢 시장·기술 분석가가 답변
"현대차 최근 뉴스 정리해줘"       →  🔵 뉴스·심리 분석가가 답변
"네이버 재무제표 분석해줘"        →  🔴 펀더멘털 분석가가 답변
```

> 💡 종목 이름이 헷갈리면(같은 이름 회사가 여럿이면) AI가 "어떤 종목인지" 한 번 되물어봅니다.

## 🛡️ 안전장치 (이 도구가 지키는 약속)

초보자가 잘못된 정보로 오해하지 않도록, 모든 결과물에 아래 원칙이 **항상** 적용됩니다.

1. **출처 없는 숫자는 절대 쓰지 않습니다.** 모든 수치 옆에 `(출처: 데이터명, 날짜)`를 붙입니다.
2. **숫자를 지어내지 않습니다.** 데이터를 못 구하면 솔직하게 **"확인 불가"**, 떠도는 소문은 **"미확인"**으로 표시합니다.
3. **"사라/팔아라" 같은 단정은 하지 않습니다.** 판단에 필요한 근거만 정리하고, **최종 결정은 사람의 몫**으로 남깁니다.
4. **항상 "학습용"임을 명시합니다.** 리포트 맨 앞에 학습용 안내, 맨 뒤에 사용한 데이터 출처 목록을 넣습니다.

## ❓ 자주 묻는 질문 (FAQ)

**Q. 프로그래밍을 전혀 몰라도 쓸 수 있나요?**
네. 위의 명령어를 복사해서 붙여넣기만 하면 됩니다. 설치(특히 사전 요건)는 한 번만 해두면 그다음부터는 명령어만 쓰면 됩니다.

**Q. 돈이 드나요?**
이 도구 자체와 사용하는 데이터(DART, 주가 등)는 **무료**입니다.

**Q. 정말 이 분석만 보고 투자해도 되나요?**
❌ **안 됩니다.** 이 도구는 **공부·연습용**입니다. 실제 투자 결정은 반드시 전문가 상담과 본인 판단으로 하세요.

**Q. 미국 주식도 되나요?**
이 도구는 **한국 주식** 분석에 맞춰져 있습니다.
