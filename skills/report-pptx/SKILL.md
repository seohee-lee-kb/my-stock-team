# report-pptx

`reports/{종목명}.md` 종목 리서치 마크다운을 **디자인된 PPTX 리포트**로 변환합니다.
실제 변환은 `build_pptx.py`가 수행하며, 이 스킬은 그 스크립트를 올바른 입력으로 호출합니다.

## 사용 방법

1. 대상 리포트 파일을 확인합니다: `reports/{종목명}.md`. 없으면 사용자에게 종목명/경로를 한 번 되묻습니다.
2. 스크립트를 실행합니다 (작업 디렉터리는 프로젝트 루트):
   ```
   python ${CLAUDE_PLUGIN_ROOT}/skills/report-pptx/build_pptx.py reports/{종목명}.md            # PPTX만
   python ${CLAUDE_PLUGIN_ROOT}/skills/report-pptx/build_pptx.py reports/{종목명}.md --pdf       # PPTX + PDF
   python ${CLAUDE_PLUGIN_ROOT}/skills/report-pptx/build_pptx.py reports/{종목명}.md --format pdf  # PDF만
   ```
   - 출력은 같은 폴더의 `reports/{종목명}.pptx` / `reports/{종목명}.pdf`로 저장됩니다. 두 번째 인자로 출력 경로(확장자로 형식 인식)를 지정할 수도 있습니다.
   - **PDF는 PowerPoint·LibreOffice 등 오피스가 없어도 브라우저에서 바로 열립니다.** matplotlib로 PPTX와 동일한 7장 구성·디자인을 직접 렌더합니다. PPTX를 열 앱이 없는 환경에서는 `--pdf`를 권장합니다.
3. 실행 후 `생성 완료: ...` 경로를 사용자에게 알립니다. 여러 종목이면 파일별로 반복 실행합니다.

의존성: `python-pptx`, `matplotlib` (미설치 시 `pip install python-pptx matplotlib`).
한글 폰트 `맑은 고딕`(`C:\Windows\Fonts\malgun.ttf`)이 필요합니다. PDF/차트의 한글은 이 폰트를 파일 경로로 직접 등록해 렌더합니다.

## 슬라이드 구성 (항상 이 순서로 고정)

1. **표지** — 종목명 · 작성일/기준일 · 작성 관점
2. **종목 개요**
3. **재무 요약** — 표 + 최근 3개년
4. **가격 / 추세** — matplotlib 차트(기간별 변동률) + 핵심 수치
5. **뉴스 · 심리**
6. **리스크**
7. **한 줄 종합**

스크립트는 마크다운 **헤딩 제목의 키워드로 섹션을 분류**해 위 7개 슬라이드에 매핑합니다.
(개요/재무/가격·추세·차트/뉴스·심리/리스크/종합 등). 파일 내 순서와 무관하게 슬라이드 순서는 고정됩니다.

## 디자인 규약 (코드에 내장됨 — 임의 변경 금지)

- 포인트색 **KB 옐로우 `#FFBC00`**, 본문은 그레이/화이트의 차분한 금융 리포트 톤.
- 한글 폰트는 **맑은 고딕** 하나로 고정(latin/ea/cs 모두 지정 → 글자 깨짐 방지).
- **모든 수치 옆 (출처/기준일) 표기.** 본문의 `(출처: …)` 캡션을 추출해 슬라이드 하단 풋터에 표기합니다.
  → 출처 캡션 없는 수치는 슬라이드에 싣지 않도록, 리서치 md 작성 시 표/수치마다 `*(출처: …, 기준일 …)*` 를 붙입니다.
- **매수/매도·목표가 단정 표현 금지.** 스크립트가 해당 패턴을 자동 스크럽합니다. 단, 출처가 명시된 증권사 목표가 등 *보도 사실*은 본문 표현에 따라 유지될 수 있으니 md 단계에서 단정 표현을 쓰지 않습니다.
- **표 넘침 방지**: 데이터 행은 최대 8행(초과 시 '…'로 생략), 열 수가 많으면 폰트를 자동 축소합니다.

## 입력 마크다운 권장 포맷

```markdown
# {종목명} 리서치
- 기준일: 2026-06-11
- 작성 관점: 가치·펀더멘털 중심 장기(3년+)

## 종목 개요
- ...

## 재무 요약
| 항목 | 2023 | 2024 | 2025 |
|---|---:|---:|---:|
| 매출액 | ... | ... | ... |
*(출처: 사업보고서, 기준일 …)*

## 가격 / 추세
| 기간 | 변동률 |
|---|---:|
| 1주 | -14.9% |
| 1년 | +434.9% |
*(출처: FinanceDataReader, 기준일 …)*

## 뉴스 · 심리
- ...

## 리스크
- ...

## 한 줄 종합
...
```

표 외 텍스트는 불릿/문장으로 자동 정리되며, 슬라이드당 본문은 상위 항목 위주로 표시됩니다(넘침 방지).
